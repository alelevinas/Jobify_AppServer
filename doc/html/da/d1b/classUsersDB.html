Detailed Description

DB that stores all the user information (except the password and session data). A typical user json can be found at the API documentation. the json fields are: "chats" : [], "city" : "Rosario", "contacts" : [], "dob" : "1993-08-19", "gender" : "male", "name" : "Alejandro Pablo Levinas", "nationality" : "argentino", "previous_exp" : [one,two,three], "profile" : "Soy un estudiante de ingenieria en informatica que se propone............blabllbla...........", "recomendations" : [], "skills" : [ "C", "C++", "GoogleTest" ], "email" : "alelevinasATgmail.com", "username" : "alelevinas@gmail.com" }
